## Hardware Overview
- Drone - DJI Tello
- GPS Receiver GT U7 _[6g]_
- Battery - Lithium Ion Polymer Battery-3.7v 350mAh _[9g]_
- Camera - Module V2 8MP _[3g]_
- Current Converter _[1.6g]_
- Raspberry pi (zero WH) _[12g]_
- microSD
